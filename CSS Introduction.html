Welcome to CSS Introduction!

CSS (Cascading Style Sheets) is a language used to describe the look and formatting of a document written in HTML.

With CSS, you can control the colors, fonts, layouts, and other visual aspects of your web page.

Examples:

/* Change font color and size */ h1 { color: blue; font-size: 24px; } /* Add a background color to a paragraph */ p { background-color: lightgray; }
/* Add margin and padding to a div */ div { margin: 10px; padding: 20px; }
/* Add a background image to a container */ .container { background-image: url('image.jpg'); background-size: cover; } /* Add a background gradient to a section */ .section { background: linear-gradient(to right, #ff0000, #0000ff); }
